== Description ==

> This Plugin requires [Mailster Newsletter Plugin for WordPress](http://rxa.li/mailster?utm_campaign=wporg&utm_source=Piwik+Analytics+for+Mailster)

Integrates Piwik Analytics with Mailster Newsletter Plugin to track your clicks
Requires the Mailster Newsletter plugin >= 2.0 and the Piwik Analytics tracking code on the website

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
2. Activate the plugin
